Bu geliştirme sonrasında sistem performansında belirgin bir iyileşme gözlemlenmiştir. Özellikle embedding tarafının Ollama’dan OpenRouter tabanlı embedding servisine taşınmasıyla birlikte PDF ingest süresi önemli ölçüde azalmıştır. Önceki ölçümlerde ingest_pdf işlemi yaklaşık 59.7 saniye sürerken, yeni ölçümlerde aynı akış yaklaşık 9.7 saniyeye düşmüştür. Benzer şekilde soru embedding süresi yaklaşık 2.3–2.5 saniye seviyesinden yaklaşık 733 ms seviyesine gerilemiştir. Retrieval aşamasında vector search ve rerank zaten düşük maliyetli çalıştığı için asıl performans kazanımı embedding üretim tarafında elde edilmiştir. Son durumda sistemdeki ana darboğaz artık retrieval değil, OpenRouter üzerinden alınan LLM yanıt süresidir; bazı sorgularda chat completion süresi yaklaşık 19.8 saniyeye kadar çıkabilmektedir. Bu nedenle mevcut optimizasyonlar ingest ve retrieval tarafını belirgin biçimde hızlandırmıştır
